<b>Ausblenden von Streams, Spielen, Kategorien, Kanälen und Tags auf: twitch.tv</b>

<b>Funktionen</b>
- Kategorien ausblenden
- Kanäle ausblenden
- Tags ausblenden
- Stream-Wiederholungen ausblenden
- Streams aufgrund ihres Titels ausblenden
- Filterung auf der "Du folgst"-Seite kann deaktiviert werden (unter Einstellungen)
- Sichtbarkeit der "X"-Buttons umschaltbar
- Sperrliste im- und exportierbar (unter Einstellungen)
- Sperrliste synchronisiert sich automatisch mit allen Geräten (unter Einstellungen, kann deaktiviert werden)
- bequemes Ein- und Ausschalten der Erweiterung
- kompatibel mit BetterTTV (BTTV) und FrankerFaceZ (FFZ)
- unterstützt Twitchs dunklen Modus

<b>Unterstütze Seiten</b>
- Durchsuchen: Kategorien
- Durchsuchen: Live-Kanäle
- Kategorie: Live-Kanäle
- Kategorie: Videos
- Kategorie: Clips
- Startseite/Entdecken (keine Filterung im Karussell)
- Du folgst
- Seitenleiste (nur Filterung, keine Buttons zum Sperren)
- Bereiche können weiterhin via Direktlink besucht werden

<b>Hinweis bzgl. Änderungen auf twitch.tv</b>
Twitch ist bekannt dafür, Änderungen an der Website ohne weitere Ankündigungen durchzuführen, weswegen diese Erweiterung über Nacht unbrauchbar werden kann (meist dienstags). Twitch liefert außerdem unterschiedliche Canary Builds (Vorschau potenziell zukünftiger Versionen) an bestimmte/zufällige Benutzer aus, die meist Teile der Erweiterung unbrauchbar machen. Sollte die Erweiterung nicht mehr funktionieren, <a href="https://github.com/kwaschny/unwanted-twitch/issues">bitte das Problem melden</a> und die Erweiterung bis zum nächsten Update deaktivieren. Updates könnten einige Tage benötigen, da diese erst vom jeweiligen Browser Store überprüft werden müssen. Also entweder <a href="https://github.com/kwaschny/unwanted-twitch/commits/master">auf GitHub vorbeischauen</a> oder <a href="https://twitter.com/Kwaschny">mir auf Twitter folgen</a>, um über neue Versionen informiert zu bleiben. Bitte hinterlasst dafür keine 1-Stern-Bewertungen, denn das hilft weder der Sache noch meinem Gemüt. Danke. 🙂

<b>Unterstütze die fortlaufende Weiterentwicklung dieser Erweiterung</b>
Ich arbeite an diesem Projekt in meiner Freizeit. Und obwohl ich selbst twitch.tv nicht mehr sonderlich häufig schaue, möchte ich Leuten das Ausblenden von Inhalten, an denen sie nicht interessiert sind, weiterhin ermöglichen. Solange Twitch also keine eigene Lösung dafür bietet, werde ich an dieser Erweiterung arbeiten und es mit allen zukünftigen Änderungen auf twitch.tv kompatibel machen.
